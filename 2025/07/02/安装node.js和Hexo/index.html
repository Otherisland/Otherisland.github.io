1. 安装nodejs

下载地址：node.js官网

![[6e2a47dd5e18c921a18278a3746dc1c6_MD5.jpg]]

如上图，直接点击下载，完成后双击安装，安装直接默认 _next_ 就好，很简单，需要改的也就是安装位置改一下，我是在D盘新建了一个文件夹 nodejs，安装在这个文件夹里。如果要看详细到每一步的话，可以看一下这个文章：node.js 安装详细步骤教程。

安装完成可以用打开cmd检验一下是否安装成功，用 node -v 和 npm -v 命令检查版本，如下图：

![[1f69e9cd9f870d8c1a5e191f2b75c846_MD5.jpg]]

设置npm在安装全局模块时的路径和环境变量

因为如果不设置的话，安装模块的时候就会把模块装到C盘，占用C盘的空间，并且有可能安装好hexo后却无法使用，所以我们需要设置一下：

在 nodejs 文件夹中新建两个空文件夹 node_cache、node_global，如下图：

![[aa00148c8da9906f12fa7bbe7573e4dc_MD5.jpg]]

打开cmd，输入如下两个命令：

![[f93bb90017343c2eb0164461f0ea12ea_MD5.jpg]]

设置环境变量：win10系统 –> 打开控制面板 –> 系统 –>高级系统设置 –> 环境变量 ，然后在系统变量中新建一个变量名为NODE_PATH，值为D:\nodejs\node_global\node_modules，如下图：

![[1bcadc5902c2ccba8668d44830fa5826_MD5.jpg]]

![[e648ba419588dc963f20ff54d4ae95e0_MD5.jpg]]

然后编辑用户变量里的Path，将相应npm的路径改为：D:\nodejs\node_global，如下图：

![[a0785424e4b7a1133cc9bc8cec3abb41_MD5.jpg]]

![[41c4279ea76f7da1b7c4f8e64fe1b768_MD5.jpg]]

![[1775e7c4bbff2b0beb1eba53596ec538_MD5.jpg]]

2. 安装Hexo

Hexo就是我们的个人博客网站的框架，在安装之前，我们要先在GitHub上创立一个仓库，如下图：

![[f0a0cd5ccd327887cf2eef7566bd89ae_MD5.jpg]]

![[203865a7553c825ca235b8fbd5dc681e_MD5.jpg]]

点击 settings 设置：

![[8e8cac10afa022cc16a3ea194cfa80e5_MD5.jpg]]

滑到页面底端，出现如下图所示就说明已经成功了：

![[3659abca05bb0518747cedb2648dd37c_MD5.jpg]]

接下来就是安装Hexo，首先在D盘建立一个文件夹 Blog，点开 Blog 文件夹，鼠标右键打开 Git Bush Here，输入npm命令安装Hexo：

![[14f6230f4136c95a9969cd8e7ea70ab2_MD5.jpg]]

安装完成后，输入 hexo init 命令初始化博客：

![[5695e3fcf56c1e2c05867b8cae8aeae8_MD5.jpg]]

然后输入hexo g 静态部署：

![[e1f1509668f4a09c6e7125cd07f885a7_MD5.jpg]]

这时网页已经部署完成，输入 hexo s 命令可以查看：

![[afca3302c033168536e0ea99786abe16_MD5.png]]

浏览器输入 http://localhost:4000 就可以打开新部署的网页：

![[eb4f2cb9b32732cdfe28755f945dd2ff_MD5.jpg]]

看完之后 ctrl +c 停止运行服务器。
